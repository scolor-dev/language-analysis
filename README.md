# いきもの文字 翻訳器

撮影した独自文字を、50音対応表と画像照合してひらがなへ変換するブラウザアプリです。

## 使い方

1. このフォルダーでローカルサーバーを起動します。

   ```sh
   python3 -m http.server 8000
   ```

2. ブラウザで `http://localhost:8000` を開きます。
3. 写真を選ぶと、文字領域の検出と認識が自動で始まります。
4. 読み順が違う場合は「自動／横／縦」を切り替えます。
5. 検出漏れは手動で囲み、誤認識は候補またはプルダウンで修正できます。

写真と認識処理はブラウザ内に留まり、外部サーバーへ送信されません。横書き・縦書きとも、囲んだ順番が読み順になります。

自動検出は写真の暗い画素を二値化し、連結した前面図形を抽出します。候補領域を50音表の前面6種類と照合して、文字に似ていない図形を除外した後、背面を含む正方形へ拡張して通常の文字認識に渡します。

## 認識方式

濃い前面画像を母音側、薄い背面画像を子音側の特徴として分離し、`assets/kana-chart.png` から切り出した46文字のテンプレートと照合します。専用の学習済みモデルやAPIキーは不要です。

認識時は中央に最も近い濃い連結成分を主図形として採用します。選択範囲の端から入り込んだ隣接文字は除外し、主図形を基準に位置と大きさを揃えます。前面で行を判定してから背面で列を判定し、±7度の傾きも照合します。

候補表示は最高一致率に応じて切り替わります。90%以上は1文字を確定表示、最高が80%以上なら一致率80%以上の文字をすべて一致率順で表示、それ未満は上位3文字を表示します。候補をクリックするか、プルダウンから任意の文字へ修正できます。

撮影画像の傾きが大きい場合や、薄い背面画像が白飛びしている場合は候補精度が下がるため、正面・均一な明るさで撮影してください。
